# Console FB2 reader

Bash script for view fb2 in console.

Example:

```bash
$ sh fb2totxt.sh -l book.fb2.zip
```
or:

```bash
$ sh fb2totxt.sh book.fb2.zip | less
```
or:

```bash
$ sh fb2totxt.sh -o book.txt book.fb2.zip
$ cat book.txt | less
```

Support zip and gzip compression
